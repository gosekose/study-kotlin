# KOTLIN-STUDY

### 규칙

* 매주 **주말**에 스터디를 진행합니다.
* 스터디는 모두 구글 미트(비대면) 진행합니다.
* **코틀린 완벽 가이드** 책을 기준으로 삼되, 자율적으로 자료를 수집하여 내용을 **10 ~ 20분** 사이로 정리합니다.
* 발표는 당일 룰렛으로 진행합니다.
